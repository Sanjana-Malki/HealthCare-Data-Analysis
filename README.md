# 🏥 Healthcare Data Analysis Dashboard

## 📌 Project Overview
This project is a Healthcare Data Analysis Dashboard developed using Power BI on a dataset of **9,216 patient visit records** spanning **April 2019 – October 2020**. It provides insights into patient demographics, admission patterns, wait times, and department-wise performance through interactive visualizations and reports.

## 🎯 Objectives
- Analyze patient and hospital visit data.
- Monitor admission trends and patient wait times.
- Track department-wise performance and patient satisfaction.
- Identify patterns to support better decision-making.
- Create an interactive dashboard for healthcare insights.

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX
- Excel/CSV Data Sources

## 📊 Key Features
- Patient Demographics Analysis (age, gender, race)
- Admission Rate Tracking
- Department-wise Wait Time & Admission Analysis
- Patient Satisfaction Scoring
- Interactive Filters and Slicers
- KPI and Trend Visualization

## 📈 Insights Generated (calculated from the dataset)
- The dataset covers **9,216 patient visits** across 19 months, with monthly volume ranging from **431 visits (Feb 2020)** to a peak of **530 visits (Aug 2020)** — a fairly stable patient inflow with no major seasonal spikes.
- **Patients aged 0–18 made up the largest age group (22.9%)**, followed by 19–30 year-olds (15.6%); patients above 60 collectively accounted for **23.9%** of all visits.
- **General Practice received the highest referral volume (1,840 visits, ~38% of all referrals)**, more than double the next-highest department, Orthopedics (995 visits).
- The overall **patient admission rate was 50.0%** — exactly half of all visits resulted in formal admission. **Renal had the highest admission rate (53.5%)**, while **General Practice had the lowest (48.2%)**, despite handling the most patients.
- **Average patient wait time was 35.3 minutes**, with **Neurology patients waiting longest on average (36.8 min)** and **Renal patients waiting the least (34.7 min)** — a relatively narrow 2-minute spread across departments.
- Of the ~27% of visits with a recorded satisfaction score, the **average satisfaction score was 4.99/10**, with male patients (5.03) scoring marginally higher than female patients (4.96).
